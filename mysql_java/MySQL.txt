●MySQLへrootでログイン
> mysql -u root -p
パスワード:searchman


●DBの作成
> create database company_db;


●ユーザー作成
> create user suser@localhost identified with mysql_native_password  by 'spass';


●ユーザーに権限を与える
> grant all on company_db.* to suser@localhost;


●rootはログアウト
> exit;


●新ユーザ(s_user)でログイン
> mysql -u suser -p
パスワード:spass


●DBの確認
> show databases;


●使用DBの指定
> use company_db;
